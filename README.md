# Udacity Readable Project

This is a reddit-like app built with react and redux for the Udaciy React Developer nanodegree.


## Installing

* Install and start the API server
    - `cd api-server`
    - `yarn install`
    - `node server`
* Install and start the frontend application
    - `cd frontend`
    - `yarn start`
